# OtoDesks
A DJ application!

## Features
- Loading of audio files from local storage into audio players
- Play two tracks simultaneously
- Deck controls:
  - Play/pause
  - Speed up/ slow down
  - Loop
  - Mute
  - Frequency
- Searching for loaded audio files by name
